Survival Stats is a game server API for popular survival MMO games, that Display details about your gaming server with the KYE Server API. 

== Description ==

Have you ever wanted to display detailed information about your Ark: Survival Evolved gaming server direct on your wordpress website.

Survival Stats: Ark Edition pings your server using the KYE Creations API and records player data, server uptime, daily and weekly server population.

= Use our service to log player activity, active admin status & security =

By registering your use of our widget you will gain access to advanced features that our system will log data being retrieved from your server, so you can keep track of players regularity.

= Logs & Displays Player list = 

As well as displaying the current player list, registered users can also display extra data that is logged every time the widget pings your game server. This includes data such as if a player is a regular player on the server or the first time a player has played.

API Information: http://www.knowyourenemy.co.uk/game-server-api-access

Basic API Generation: http://www.knowyourenemy.co.uk/packages/basic-api-key

Advanced API Generation: http://www.knowyourenemy.co.uk/packages/advanced-api-key

Requires WordPress 4.0 + and Latest version of jQuery.

== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'Survival Stats'
3. Activate Survival Stats from your Plugins page.
4. Visit 'SS: Ark Settings' and enter' and enter your API and server details. 
5. Add the [ssark] shortcode to all pages you want to display that server stats on. 

= From WordPress.org =

1. Download Survival Stats: Ark Edition.
2. Upload the 'bbpress' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate Survival Stats from your Plugins page.
4. Visit 'SS: Ark Settings' and enter' and enter your API and server details. 
5. Add the [ssark] shortcode to all pages you want to display that server stats on.

= Extra =

1. Visit 'SS: Ark Settings' and enter your API Token, Public Key, Secret Key that can be obtained from - knowyourenemy.co.uk.
2. Enter your Ark gaming server connection details, using the rcon port and not the query port.
3. Add the [ssark] shortcode to all pages you want to display that server stats on,using the id attribute to display multiple servers* Available in pro with advanced API key.

== Frequently Asked Questions ==

= How many servers can I display on my WordPress site? =

Current version of Survival Stats only allows one server to be displayed, PRO version of the plugin is due for release very soon and will allow up to 5 servers on a advance API key.

= Can I change the colours of the server stats display? =

A future release will allow customisation from the admin console, we are currently working on this for next update of the plugin.

== Upgrade Notice ==

Upgrade is not required to unlock additional server display details, please purchase an advanced API key from the KYE Creations website - http://www.knowyourenemy.co.uk
